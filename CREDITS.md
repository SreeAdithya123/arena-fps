# Asset credits & licenses

Every asset in DEPOT is generated by the project's own code at runtime.
There are **no third-party art, model, font, or audio assets**.

| Category | Source | License |
|---|---|---|
| Textures (concrete, metal, crates, walls, flashes, decals) | Procedural `<canvas>` generation — `src/render/renderer.js`, `effects.js` | Project license |
| 3D models (arenas, soldiers, weapons) | Primitive-built three.js geometry — `src/sim/maps/*`, `src/render/*` | Project license |
| Audio (weapons, footsteps, UI, stingers, ambient beds) | Synthesized WebAudio — `src/render/audio.js` | Project license |
| Fonts | System font stack (Segoe UI / system-ui) — nothing bundled | n/a |
| Map thumbnails | In-engine captures of this project's own maps | Project license |

Runtime dependency: [three.js](https://github.com/mrdoob/three) (MIT).
Build tooling: Vite (MIT), Wrangler (Apache-2.0/MIT).

Game design is original. DEPOT shares a genre with deadshot.io but copies
none of its assets, art, code, maps, or text.
